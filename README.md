iuri.blog.br

Código do site iuri.blog.br

# Dependências:

Perl versão 5;
Servidor Web;
Servidor Web com capacidade para trancar diretórios com senha para usar o admin/;

## Pacotes perl/cpan:

CGI
DateTime::Format::Mail
Email::Date::Format
File::Slurp
IO::File
XML::RSS
